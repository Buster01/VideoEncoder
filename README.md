# VideoEncoder

## Features
* Inputfiles: *.ts, *.mkv
* Videocodec Software: h.264, h.265, copy
* Videocodec Hardware (bei geeigneter Hardware): 
    * Intel Quick Sync h.264 und h.265
    * NVidia Encode h.264 und h.265
* Audiocodec copy, AAC
* Videodecoder: DXVA oder Software
* Deinterlace: yadif
* Outputfiles: *.mkv


## Build
Visual Studio 2017
.Net Framework 4.6.1
